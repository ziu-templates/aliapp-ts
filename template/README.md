### 使用说明

#### 指令说明

- 启动本地开发

```javascript
npm run dev
```

- 打包`testing`环境代码

```javascript
npm run testing
```

- 打包`staging`环境代码

```javascript
npm run staging
```

- 打包`production`生产环境代码

```javascript
npm run build
```

- 依赖包分析

```javascript
npm run report
```

- 使用`angular commit message`提交`git commit`信息

```javascript
npm run commit
```

- 安装`git hook`

```javascript
npm run husky
```

- 卸载`git hook`

```javascript
npm run unhusky
```
